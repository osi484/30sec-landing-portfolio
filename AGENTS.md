# '30sec' Landing Page - AGENTS.md

## 기술 스택
- Vite + React (TypeScript)
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (Animations - Optional but Recommended)

## 코드 스타일 및 규칙
- 모든 컴포넌트는 `src/components/` 산하에 기능별 폴더로 관리할 것.
- 색상 팔레트: `#000000` (Deep Black), `#FF0000` (Urgent Red), `#1A1A1A` (Dark Grey) 등을 기조로 함.
- 폰트: 'Inter' 또는 'Space Grotesk' 등 현대적이고 긴장감 있는 산세리프 폰트 권장.
- 사전등록 버튼은 `Glassmorphism` 효과와 호버 시 강렬한 강조 효과를 포함할 것.
